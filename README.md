## Introduction

RPSCC project is a robust parameter server with configurable consistency.

## Installation

```sh
git clone https://github.com/purkyston/rpscc
cd rpscc
mkdir build & cd build
cmake ..
make
```

This scripts will automatically generate protobuf files.
